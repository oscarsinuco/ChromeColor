It is a function that returns a promise that when resolved returns an HTML Object.

Its possible parameters are:
removecolors(id | url , colors , variability)

id | url: #id reference to the id of the html object that contains the image or url of the image.

colors: A string or Array that represents the color to be removed, can be red, yellow, blue, green, black, white or orange. Example: 'blue' or ['blue', 'red']

variability: A double value that must be >= 0 and < 1 represents the variation of the color to be removed.


import { removeColors } from "./modules/colors-extract.min.js";
removeColors('#imagen', ['black'], 0.5).then( canvas => {
  document.querySelector(".example").appendChild(canvas)
}).catch( err => {
  console.log(err)
})
